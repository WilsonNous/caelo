# CAELO — Portal (HTML/CSS/JS)

Portal institucional (MVP) da CAELO Consultoria e Assessoria, com animações e transições suaves.

## Estrutura
- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/logo.png` (adicione o logo aqui)

## Rodar local
Basta abrir `index.html` no navegador.

Se quiser rodar com servidor local:
- VSCode: Live Server
ou
- Python:
  - `python -m http.server 8000`

## Deploy no Render (Static Site)
1. Suba o repo no GitHub.
2. No Render: **New → Static Site**
3. Conecte ao repositório
4. Configure:
   - Build Command: *(vazio)*
   - Publish Directory: `.`
